## What you will do:

**Problem Statement**  
Scenario-Based Problem Statement

Project Title: Creating Your First Python Greeting Script for Company-Wide Onboarding

Role & Context:
You have just started your journey as a Junior Application Developer at a growing technology solutions company. Every time a new employee joins, your company encourages developers to set up a welcome message as a classic introduction to its Python-centric culture. You have been assigned the essential task of creating a simple Python script that displays a personalized "Hello, World!" greeting in the console. This foundational exercise is critical, as it ensures all developers can write, save, and run basic scripts—skills that form the backbone of all future projects in the company.

Objective:
Develop a Python script that outputs a greeting message to the console using the print statement. You will learn to write and save this script in a file, then execute it to verify its output. This exercise not only introduces the Python print syntax but also reinforces the fundamental workflow of writing, saving, and running Python programs.

Task Overview & Relevance:
As the onboarding developer, your goal is to create a script named hello_world.py. This script will generate the message:

Hello, World! Welcome to [Your Company Name].

all within the console window. This simple application will be used as a test to verify that every developer has a working Python environment and understands the basic process of script execution—skills demanded daily in real-world development environments.

Learning Outcomes:

- Demonstrate the ability to write, save, and run a basic Python script on your local machine.
- Accurately recognize and implement the Python print statement syntax in a script.
- Gain confidence in using the command line or terminal for Python application execution, a foundational step in real-world application development.

Target Audience Alignment:
This project is specifically intended for beginners—individuals new to Python programming or software development. It assumes:

- No prior programming experience beyond installing Python on their computer.
- No advanced knowledge of frameworks, libraries, or application architectures.
- Comfort with following step-by-step instructions and using basic operating system features (e.g., file system navigation).

Time Constraint:
The entire task must be completed within 15 minutes, reflecting the simplicity and foundational focus of a hello_world project. This timeline ensures learners remain focused and achieve instant, confidence-boosting results.

Project Execution Breakdown (Step-by-Step):

1. Create a New Python File (2 minutes)
   - Open any basic text editor (such as Notepad, VS Code, or Sublime Text).
   - Create a new file and name it hello_world.py.

2. Write the Greeting Script (3 minutes)
   - In hello_world.py, type the following line exactly:
   
     ```python
     print("Hello, World! Welcome to [Your Company Name].")
     ```
   - Replace [Your Company Name] with the actual name (e.g., print("Hello, World! Welcome to TechNova.") ).

3. Save the Script (2 minutes)
   - Save the file in an easily accessible directory (e.g., Desktop or a dedicated Projects folder).

4. Run the Script Using the Console (3 minutes)
   - Open a terminal (Command Prompt on Windows, Terminal on macOS/Linux).
   - Navigate (cd) to the directory where hello_world.py is saved.
   - Execute the script by entering:

     ```
     python hello_world.py
     ```
   - Observe the output in the console window.

5. Verify Output (2 minutes)
   - Ensure that the following message is displayed in the console:

     Hello, World! Welcome to [Your Company Name].

   - If it matches, you have successfully completed the task.

6. Reflect on What You've Learned (3 minutes)
   - Consider how this process (writing, saving, running scripts) forms the bedrock of all future Python development.
   - Discuss with a peer or note down the purpose of the print statement and its syntax.

Deliverable:
A file named hello_world.py, executed in the console, successfully displaying the required greeting message.

Expected Features & Boundaries:

- The application must use only core Python (no external libraries).
- Output must be displayed in the console via the print statement.
- Only single-file, straightforward script structure (“Simple application structure”).
- No advanced logic, input handling, or program branching.

Assessment Criteria:

- The script can be saved and run independently.
- The output precisely matches specifications.
- The learner can describe (orally or in writing) the print statement’s role and syntax in Python.

By following these steps, you demonstrate essential industry readiness for Python-based application development and set a successful foundation for more complex projects ahead.
